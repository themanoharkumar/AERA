# AERA – AI Emergency Response Assistant

AERA is an AI-powered emergency monitoring platform that analyzes CCTV and webcam feeds to detect emergencies such as fire, smoke, falls, intrusion, and other suspicious activities.

## Features (Planned)

- Multi-camera monitoring
- Fire detection
- Smoke detection
- Fall detection
- Intrusion detection
- AI-generated incident reports
- Evidence capture
- Threat assessment
- Analytics dashboard
- Real-time alerts

## Tech Stack

- Python
- OpenCV
- PyTorch
- YOLO
- FastAPI
- Streamlit
- NumPy
- Pandas
- scikit-learn
- spaCy
